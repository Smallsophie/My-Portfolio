# Sofia Wanjiku — Developer Portfolio

A modern personal portfolio built to showcase my projects, skills, and development progress as a front-end developer.

This website serves as a central hub for my work, featuring interactive UI elements, smooth animations, and a structured layout designed for clarity and usability.

## Features

- Responsive, mobile-first layout
- Interactive terminal-style hero section
